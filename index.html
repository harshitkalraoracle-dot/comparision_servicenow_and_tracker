Setup Guide

LZ-String compression — links are self-contained, no Apps Script needed for link generation. Apps Script only used for collecting responses.

1

Create a Google Sheet

Go to sheets.google.com → create blank spreadsheet → name it Incident Tracker Responses.

2

Open Apps Script

Extensions → Apps Script → delete all code → paste entire contents of Code.gs → Save.

3

Deploy as Web App

Deploy → New deployment → Web app
Execute as: Me · Access: Anyone → Deploy → Authorize → Copy the URL.

https://script.google.com/macros/s/AKfycb.../exec
4

Update form.html

Open form.html → find const APPS_SCRIPT_URL = "..." → paste your Web App URL → save.

5

Update index.html

Open index.html → find const GITHUB_PAGES_BASE = "..." → paste your GitHub Pages URL (no trailing slash) → save.

6

Push to GitHub & enable Pages

Push all 3 files → Settings → Pages → Source: main / root → Save. Live in ~60 seconds.

7

Paste Apps Script URL in topbar

On your live site, paste the URL in the Apps Script URL topbar field. This is only used for the Responses tab (to read back submissions).

8

Daily workflow

1. Upload both Excel files → Analyze.
2. Click team member card → link generates instantly (no network call).
3. Copy & send via WhatsApp/Teams.
4. They fill & submit → Responses tab → Refresh → Export CSV.
