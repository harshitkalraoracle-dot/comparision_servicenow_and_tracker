Setup Guide

LZ-String compression — links are self-contained.

1

Create a Google Sheet

Go to sheets.google.com → name it Incident Tracker Responses.

2

Open Apps Script

Extensions → Apps Script → paste Code.gs → Save.

3

Deploy as Web App

Execute as: Me · Access: Anyone → Deploy → Copy URL.

https://script.google.com/macros/s/AKfycb.../exec
4

Update form.html

Find const APPS_SCRIPT_URL = "..." → paste your Web App URL.

5

Update index.html

Find const GITHUB_PAGES_BASE = "..." → paste your GitHub Pages URL.

6

Push to GitHub & enable Pages

Settings → Pages → Source: main / root → Save.

7

Paste Apps Script URL in topbar

Used only for the Responses tab to read back submissions.

8

Daily workflow

Upload both Excel files → Analyze → Click team member → Copy link → Send → Responses tab → Export CSV.
